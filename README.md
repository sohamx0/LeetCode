# LeetCode Solutions

## Purpose
This repository is a clean, beginner-friendly collection of my LeetCode solutions, organized by topic and difficulty for easy learning and review.

## Folder Structure
Solutions are grouped by **topic**, then by **difficulty**:

```
<Topic>/
	Easy/
	Medium/
	Hard/
```

Example file name:
```
LC_1_Two_Sum.py
```

Topics included:
Arrays, Strings, Linked List, Stack, Queue, Recursion, Backtracking, Trees, Graphs, Greedy, Dynamic Programming, Binary Search, Math.

## Languages Used
- Python
- C++

## How to Use This Repo
- Pick a topic you want to practice.
- Start from Easy, then move to Medium and Hard.
- Open a solution file to see the approach summary and complexity.
- Re-implement the solution on your own for learning.
